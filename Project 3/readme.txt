## BuddySystem.java
- simulates the buddy system memory allocation algorithm
- addMem: allocates memory 
- freeMem: free blocks
- merge: merges blocks if they are free

## MemoryBlock.java (static class)
- used to store information about memory blocks (process names, size, address, etc.)

## How to Compile & Run the Program
javac BuddySystem.java
java BuddySystem
